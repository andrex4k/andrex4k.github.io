Изменение приветсвия при логине в Bash

5 ноября 2017 - 1 minute read -

Редактируем файл /etc/bash.bashrc, для изменения приветсвия при логине от root.

### ===============================================
### Простой сценарий bash для входа в систему перед открытием терминала.
### ===============================================

while true; do
  # Don't exit at Ctrl-C
  trap "echo" SIGINT

  printf "\n"
  echo -n " Ты кто такой, друг? "; read -s name;
  if [ "$name" == "name" ]; then
    reset
    printf "\n Добро пожаловать мой КОРОЛЬ!\n"
    printf " "; date;
    printf " Не забудь обновить систему.\n\n"
    break
  else
    printf "\n Эй ты! кто такой!! Ты не владелец этой машины!\n"
  fi
done
